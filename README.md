# Landing page do curso Razor Pages

Landing page em React/Vite para o curso ASP.NET Razor Pages.

## Stack
- Vite + React + TypeScript
- Tailwind CSS e shadcn/ui

## Desenvolver
```sh
npm install
npm run dev
```

## Build
```sh
npm run build
```

## Deploy
O build estático é gerado em `dist/`. Publique esse diretório no provedor de sua escolha ou use `npm run deploy` para publicar via GitHub Pages.
